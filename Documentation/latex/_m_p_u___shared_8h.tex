\doxysubsection{Detailed Description}
Shared functions and macros. 

This file includes all the function prototypes, macros, and needed header files for the MPU\+\_\+\+Shared.\+c which has the definitions of functions used by all the other source files of the MPU6050 library. It does not need to be included as all other files include it.
