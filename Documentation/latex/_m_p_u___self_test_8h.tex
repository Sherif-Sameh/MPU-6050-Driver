\doxysubsection{Detailed Description}
For testing the relaibility of the IMU\textquotesingle{}s readings. 

This file includes all the function prototypes and macros for using the MPU6050\textquotesingle{}s self-\/test feature which can be used to insure that the device can still deliver relaible readings. \begin{DoxyAttention}{Attention}
According to the manufacturers\textquotesingle{}s datasheet, the FS range for testing the gyros and accelerometers should be +-\/250 dps and +-\/8g respectively.
\end{DoxyAttention}
The Change from the factory trim is calculated as follows\+:  \[
  \textnormal{Change from the factory trim} = \frac{STR - FT}{FT}
\] Where STR is the self-\/test response and equals to\+: $ STR = (\textnormal{Output with self test enabled} - \textnormal{Output with self test disabled}) $ ~\newline
 The acceptable change is within +-\/ 2\%.
