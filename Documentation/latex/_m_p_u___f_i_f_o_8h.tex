\doxysubsection{Detailed Description}
For using the internal FIFO on the MPU6050. 

This file includes all the needed function prototypes macros to be able to use the internal FIFO for storing and readings the sensors\textquotesingle{} output values. \begin{DoxyAttention}{Attention}
The size of the FIFO is 1024 bytes. 
\end{DoxyAttention}
